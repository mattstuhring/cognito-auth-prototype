# AWS Cognito Auth Prototype

## Description

Backend REST API for auth using AWS Cognito SDK.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# Development
$ npm run start:dev

# Production
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
